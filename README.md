## 🚀 Featured Project

### [Brazil Weather Pipeline](https://github.com/SrEdward/brazil-weather-pipeline)

End-to-end data platform ingesting historical and real-time weather data for 10 Brazilian cities.

- **Batch layer:** Airflow-orchestrated ELT → AWS S3 → Snowflake → dbt (staging + marts)
- **Lakehouse layer:** Apache Iceberg on S3, catalogued via AWS Glue, queried with DuckDB
- **Streaming layer:** Kafka producer/consumer writing real-time events to Iceberg
- **IaC:** Full infrastructure provisioned via Terraform
